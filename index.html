    PRESENTATION OF OUTPUT

    AUMOVIO Central Hub Web Development

    This is the project I created where when the user clicks the service, it will direct them automatically to that link. Designed as an internal portal dashboard, it functions as a centralized utility navigation matrix allowing personnel to trigger instant redirection paths to external tools, identity management spaces, and organizational link indexes.

    Network Optimization & Operational Compliance

    When a user's laptop has no Network connection, this is the solution utilized to resolve the issue. The workflow involves enabling PC on Active Directory, executing local network driver updates to repair broken wireless/wired stacks, deploying infrastructure patches to rectify system connectivity conflicts, and running Group Policy updates (gpupdate) to re-sync security policies with the central domain controllers.

    System Re-Imaging (Re-DS Deployment)

    The first step is to click and click F12 for Dell devices and F9 for HP machines. Executing this hardware-level key command during boot forces the hardware layer to initialize the boot override system, which directly prompts this interface display. From this configuration view, selecting the Flash Drive option triggers the targeted system initialization sequence to start the clean Re-DS layout setup.

    BitLocker Security Recovery & Identity Support

    Users often come to the office to ask for support when there's a BitLocker lockout issue. To resolve this encryption challenge, we navigate through the centralized enterprise administration platform into Active Directory, locate the user's explicit device configuration object name, extract the dynamically archived BitLocker Recovery password, and manually enter that security key into the user's locked workstation interface to successfully restore operating system access.
